## 四、各屏详细说明

### ① 登录 / 注册

- **目标**：身份接入 + 协议授权
- **核心元素**：手机号输入、6 位验证码、登录按钮、3 个第三方登录（微信/邮箱/Apple）、用户协议
- **关键逻辑**：手机号验证码登录（默认）；首次登录直接进入 ⑳ 新人资料填写
- **商业意义**：合规获取手机号便于后续短信召回 + 防多账号注册

### ② 用户列表（发现）

- **目标**：核心曝光位，撮合用户与潜在 CP
- **核心元素**：5 个 Tab（推荐/附近/新人/在线/动态）、筛选标签（性别/年龄/同城/认证）、用户卡片（头像 + 在线状态 + 年龄距离 + 兴趣标签）
- **关键逻辑**：默认推荐 Tab 基于兴趣画像 + 地理位置；用户卡片点击进入 ⑤ 个人主页
- **商业意义**：VIP 用户拥有「高级筛选」「优先曝光」特权，提升曝光效率

### ③ 消息列表

- **目标**：关系沉淀 + 高频转化入口
- **核心元素**：顶部 4 个快捷入口（喜欢我/访客/礼物/通知，含红点未读数）、私聊列表（头像 + 名字 + 最后一条 + 时间 + 未读数）
- **关键逻辑**：「喜欢我」「访客」需 VIP 才能查看，构成核心付费钩子
- **商业意义**：未读 Badge 是 VIP 转化最强触发器，对应 ⑮ VIP 开通

### ④ 个人中心

- **目标**：自我管理 + 数据展示
- **核心元素**：顶部用户头部（头像 + ID + VIP 标 + 签名）、4 项数据（关注/粉丝/好友/获赞）、VIP 横幅、菜单分组（我的喜欢/谁看过我/我的礼物/我的动态 / 钱包/认证/设置/反馈）
- **关键逻辑**：「谁看过我」「我的喜欢」均为付费导流入口
- **商业意义**：聚合所有付费触点，是承接 ⑨ 钱包、⑮ VIP 等关键页面的中转站

### ⑤ 个人信息展示

- **目标**：用户名片主页，承载付费交互
- **核心元素**：大封面图、资料卡（昵称/性别年龄/VIP/ID/距离/活跃时间/标签/签名）、基本信息（身高/体重/职业/学历/家乡/情感）、兴趣爱好、九宫格相册、底部 3 个 CTA（送礼 + 关注 + 打招呼）
- **关键逻辑**：「打招呼」即开启 ⑦ 聊天对话；「送礼」即调起 ⑧ 礼物面板
- **商业意义**：单点商业化密度最高的页面，3 个 CTA 全部为付费触发

### ⑥ 充值（钻石）

- **目标**：人民币 → 虚拟币（钻石 💎），是所有付费动作的前置依赖
- **核心元素**：钻石余额、6 档套餐（60/300/680/1280/3280/6480 💎，对应 ¥6/30/68/128/328/648）、首充翻倍、3 种支付方式（微信/支付宝/Apple Pay）、虚拟币说明
- **关键比例**：`1 元 = 10 钻石`（基准）
- **关键逻辑**：套餐分档严格按消费心理学定价；「超值热销」用 68 元锚定中等消费用户
- **商业意义**：所有「⑦ 聊天扣费」「⑧ 礼物」「⑯ 视频通话」消耗，最终都回归到此页

### ⑦ 聊天对话（消费扣费）

- **目标**：核心消费场景 1 ——文字聊天付费
- **核心元素**：顶部聊天信息栏 + VIP 标、**钻石余额条**（实时显示余额 + 单条单价 + 充值入口）、消息流（系统提示 / 文字气泡 / 礼物气泡 / `- 1💎` 扣费小标）、底部输入区 + 工具栏（礼物/照片/视频通话/位置/红包）
- **关键逻辑**：普通用户每条 1💎，VIP 用户「每日 100 条免费」，至尊 VIP 无限
- **商业意义**：高频低单价消费，培养用户付费习惯

### ⑧ 礼物赠送面板

- **目标**：核心消费场景 2 ——单笔高客单消费
- **核心元素**：半透明遮罩 + 底部弹出礼物面板、4 个分类 Tab（热门/浪漫/豪华/背包）、8 款礼物（66 ~ 5200 💎）、数量选择（×1/×10/×99/×520）、底部余额 + 充值 + 送出按钮
- **关键逻辑**：礼物价值 = 平台分成 70% + 主播分成 30%（可调）；高价礼物（戒指 520、跑车 1314、城堡 5200）瞄准重度付费 / 炫耀场景
- **商业意义**：单笔流水大，是 ARPPU 主力来源

### ⑨ 钱包（收益总览）

- **目标**：收益侧账本，主播 / 高人气用户的核心页面
- **核心元素**：可提现人民币余额、4 个快捷操作（提现/充值/银行卡/明细）、本月收益（拆解为礼物/聊天/视频通话）、最近账单流水（收入红色 / 支出灰色）
- **关键比例**：钻石 → 人民币提现比例 `10 钻石 = 0.6 元`（即平台抽佣 ~40%）
- **关键逻辑**：本月收益清晰拆解，激励用户冲业绩
- **商业意义**：保留高净值用户的关键页面

### ⑩ 提现（人民币）

- **目标**：虚拟资产 → 真实现金
- **核心元素**：可提现余额 + 全部按钮、金额输入 + 快捷档位、手续费规则（0.6%）+ 实际到账显示、3 种到账方式（微信钱包 / 支付宝 / 银行卡）、限额规则、实名提示
- **关键规则**：
  - 单日最多 3 次，每日总额 ¥5000
  - 单笔 ≥ ¥10，手续费 0.6%（最低 ¥1）
  - 首次提现需 ⑰ 实名认证
- **商业意义**：合规出口 + 风控关卡

### ⑪ 邀请主页

- **目标**：邀请裂变核心入口
- **核心元素**：Hero 标语「邀好友赚 ¥3,888」+ 3 项数据（已邀请 / 已充值 / 累计收益）、邀请码 + 复制、5 个分享入口（微信/朋友圈/QQ/海报/链接）、**双向奖励**（你 388💎 / TA 288💎+7天VIP）、4 阶任务进阶、CP 组队邀请、特权解锁进度条（3/10/20/50/100 人 5 档）
- **奖励机制**：
  - 双向钻石奖励（拉新基础）
  - 首充 30% 现金返佣（质量保障）
  - 邀请人数解锁特权（长期粘性）
  - CP 组队（差异化玩法）

### ⑫ 邀请排行榜

- **目标**：通过头部用户驱动持续拉新
- **核心元素**：周榜 / 月榜 / 总榜 Tab、¥128 万现金奖池 + 倒计时、前三领奖台（冠军 ¥58888）、4~10 名榜单（头像 / 邀请数 / 奖励）、悬浮「我的排名」（#28 · 差 4 人）
- **商业意义**：用大额奖池刺激强裂变用户长期投入

### ⑬ 我的战绩

- **目标**：邀请数据可视化，引导持续优化
- **核心元素**：累计收益 ¥686.50、6 宫格数据（已邀 / 已实名 / 已充值 / 钻石 / 返佣 / 特权奖金）、状态筛选 Tab、邀请人列表卡片（每张含状态 pill + 进度条 33/66/100% + 单笔奖励）
- **商业意义**：让用户清晰看到「再推动一步即可拿钱」，提升回访拉新

### ⑭ 新人邀请落地页

- **目标**：被邀请人首次打开链接的承接页，转化为注册用户
- **核心元素**：邀请人头像背书、「价值 ¥168 等你领」主视觉、**6 件见面礼**（钻石 / VIP / 玫瑰 / 免费聊 / 心动卡 / 优惠券）、24 小时倒计时、6 大功能卖点、一键领取注册按钮
- **关键策略**：礼包总价 ≥¥150 才能跨过用户决策门槛；倒计时制造紧迫感

### ⑮ VIP 开通页

- **目标**：订阅型收入
- **核心元素**：深紫金背景调性、用户当前 VIP 状态、**月卡 ¥30 / 季卡 ¥68（限时 5 折）/ 年卡 ¥198**、8 大特权宫格、**普通 / VIP / 至尊**三档对比表、自动续费协议
- **商业意义**：稳定的订阅型现金流（MRR），LTV 提升关键

### ⑯ 视频通话计费

- **目标**：高 ARPU 实时交互场景
- **核心元素**：全屏对方人像 + 自己小窗、右上角通话时长 + **6💎/分计费**（VIP 8 折，至尊 5 折）、余额警告条（"剩 8 分钟"）、礼物弹幕、底部 5 按钮（静音 / 翻转 / 送礼 / 聊天 / 挂断）、底部余额 + 已消耗 + 充值条
- **关键逻辑**：每分钟实时扣费，余额 < 60💎 强弹充值；通话中支持送礼
- **商业意义**：单分钟 ARPU 最高的场景，且付费频次高

### ⑰ 实名认证

- **目标**：合规 + 提现 / 视频通话卡点
- **核心元素**：3 步进度条（填写 → 人脸 → 完成）、姓名 / 证件类型 / 证件号 / 出生日期、身份证正反面上传、人脸识别框（带扫描动效）、认证须知
- **触发场景**：
  - 首次 ⑩ 提现
  - ⑯ 视频通话前
  - 单日聊天 > 100 条（防黑产）
- **合规意义**：满足《互联网用户账号信息管理规定》、未成年人保护

### ⑱ 邀请海报弹层

- **目标**：分享出去的视觉载体
- **核心元素**：黑色遮罩 + 海报卡片（邀请人头像 + 标语 + 4 件礼包 + 二维码 + 邀请码）、5 大分享渠道（微信 / 朋友圈 / QQ / 链接 / 保存图片）
- **商业意义**：朋友圈裂变核心载体，海报美观度直接决定分享转化率

### ⑲ 动态广场 Feed

- **目标**：内容生态，提升 DAU 与时长
- **核心元素**：关注 / 推荐 / 附近 / 话题 Tab、热门话题胶囊、用户动态卡片（图文 + 标签 + 点赞 / 评论 / **送礼** / 分享）、底部 4 Tab 导航（含新增「动态」）
- **商业意义**：内容是私聊的破冰素材，且动态可送礼 = 即时变现入口

### ⑳ 新人资料填写

- **目标**：新用户引导，资料完整度直接决定后续匹配质量
- **核心元素**：5 步进度条、头像上传（真人照通过率 +80% 提示）、性别二选一、基本信息（昵称 / 生日 / 城市 / 身高 / 职业）、12 个兴趣标签（选 3-8 个）、跳过入口
- **关键逻辑**：头像 = 转化核心；性别决定后续推荐池；兴趣标签用于推荐算法
- **商业意义**：资料越完整，匹配率越高，付费转化率越高

### ㉑ 首充礼包弹窗

- **目标**：新人首次充值转化漏斗
- **核心元素**：全屏遮罩 + 炫彩卡、🎉 顶部装饰、24h 倒计时、两档套餐（300💎 / 680💎，均翻倍赠送）、加赠豪华礼包（VIP / 玫瑰 / 心动卡 / 优先曝光）、一键购买按钮
- **触发时机**：用户首次想发起付费动作时（如送礼、视频通话、查看访客）
- **商业意义**：把"不付费用户"变成"付费用户"的关键转化节点

### ㉒ 设置 & 安全中心

- **目标**：合规 + 隐私 + 用户留存
- **核心元素**：账号安全（手机号 / 实名 / 密码 / 设备）、隐私设置（隐身访问 / 隐藏位置 / 仅 VIP 可联系 / 黑名单）、消息通知开关、服务与帮助（客服 / 举报 / 反馈 / 协议 / 关于）、退出 / **注销账号**
- **合规意义**：满足"账号注销"、"个人信息删除"等监管要求

## 五、商业化逻辑

### 5.1 收入构成

| 收入类型 | 占比预估 | 触发页面 |
|---|---|---|
| 礼物分成 | 40% | ⑧ 礼物面板 |
| 视频通话计费 | 25% | ⑯ 视频通话 |
| 聊天扣费 | 15% | ⑦ 聊天对话 |
| VIP 订阅 | 12% | ⑮ VIP 页 |
| 首充活动 / 其他 | 8% | ㉑ 首充弹窗 |

### 5.2 用户成长漏斗

```
新人落地页 (⑭) → 安装注册 (① → ⑳)
                       ↓ 实名 (⑰)
                       ↓
发现推荐 (②) → 个人主页 (⑤) → 打招呼聊天 (⑦)
                                       ↓
                                  首充弹窗 (㉑) ← 关键转化节点
                                       ↓ 充值 (⑥)
                                       ↓
                            送礼 (⑧) / 视频通话 (⑯)
                                       ↓
                                 VIP 开通 (⑮) ← 高净值留存
                                       ↓
                              邀请裂变 (⑪⑫⑬⑱)
```

### 5.3 关键转化指标（建议）

| 指标 | 目标值 | 计算口径 |
|---|---|---|
| 注册转化率 | ≥35% | 落地页打开 → 注册 |
| 资料完整率 | ≥70% | 注册 → 完成 5 步资料填写 |
| 实名认证率 | ≥30% | 注册 → 完成实名 |
| 首充转化率 | ≥8% | 注册 → 首次充值 |
| 付费用户占比 | ≥15% | 累计付费用户 / 累计注册 |
| ARPPU（月） | ≥¥80 | 月付费金额 / 月付费用户数 |
| VIP 渗透率 | ≥10% | VIP 用户数 / 月活 |
| 7 日留存 | ≥25% | 注册 7 日后回访 |

### 6.1 钻石（虚拟币）规则

- 用途：聊天 1💎/条、视频 6💎/分、礼物 66~5200💎、私信解锁等

### 6.2 收益与提现规则

- 单日提现 ≤ 3 次，总额 ≤ ¥5000

- 提现到账 1-3 工作日
